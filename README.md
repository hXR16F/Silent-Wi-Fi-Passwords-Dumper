# Silent Wi-Fi Passwords Dumper
Simple batch file script that dumps all saved Wi-Fi passwords on local PC.

# Usage
Launch:
> src/Silent-Wi-Fi-Passwords-Dumper.bat

It will save passwords into text file:
> dump.txt
